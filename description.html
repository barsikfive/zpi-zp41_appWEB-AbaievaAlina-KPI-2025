Опис предметної галузі та бізнес-логіки

Тема: веб-додаток сервісу цифрової дистрибуції відеоігор.

Опис бізнес-логіки:

  1. Головна сторінка містить каталог цифрових товарів для дистрибуції.
  2. Розділ FEATURED анімований та стилізований під актуальні події, фестивалі або акції.
  3. Розділ ON SALE стилізований для виведення інформації про топ товарів з великою знижкою (від 50%).
  4. Розділ TOP SELLERS пропонує користувачу найгарячіші позиції.
  5. Меню каталога має бути пов'язане з іншими сторінками сервісу.
  6. Користувачу пропонується оформлення підписки на особливі пропозиції та акції.
